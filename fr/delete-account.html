Delete account

Important: If you keep the email-only web path, make sure you monitor legal@skydivewinds.com. For Google Play review, a dedicated web form may be even safer in the future.

In-app deletion path

  1. Open Profile.
  2. Go to the account section.
  3. Tap Delete account.
  4. Confirm the deletion flow.

The in-app deletion path should be the main way to delete the account when you still have access to the app.

Alternative web request if you cannot access the app

If you no longer have access to the app, email legal@skydivewinds.com with the subject Account deletion request and include the email address linked to your account.

This page is intended to serve as the web resource referenced for account-deletion purposes.

What is normally deleted

  • Authentication account and linked profile data.
  • Jump log entries linked to the account.
  • Favourite, home and recent dropzone links linked to the account.
  • Notification tokens linked to the account, where reasonably possible.
  • Other account-linked personal data no longer required for legal, security or technical reasons.

What may remain for a limited time

  • Data retained temporarily in backups.
  • Security, anti-abuse or diagnostic records needed for a limited period.
  • Data that must be kept to comply with legal obligations, if any apply.

If some information must be retained, it should be limited to what is necessary and for no longer than required.

Timing and confirmation

Deletion requests should be completed within a reasonably quick period. If processing takes additional time, users should be informed. Where feasible, a confirmation message should be sent once the request has been completed.

Need help?

If you have trouble using the in-app deletion flow or need help identifying the account, contact support@skydivewinds.com. Privacy-related deletion requests may also be sent to legal@skydivewinds.com.
